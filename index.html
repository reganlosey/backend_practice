Thanks for visiting my API

This server hosts a variety of coffee and tea objects

    The endpoints are:
  • GET: /brews. Returns an array of all coffee and tea objects
  • GET: /brews/:id. (Where id is the id of an object in the array) Returns a coffee/tea object with matching id
  • POST: /brews

There is currently no database in place to save POST requests. I'm working on it, but I'm a frontend dev and still learning about all this cool backend stuff:)
